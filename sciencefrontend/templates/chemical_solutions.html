README:

This script automates the calculation for creating solutions from dry chemicals (e.g. powders) of a certain molarity. For example, if you wanted to create a 1M solution of chemical X, you would want to know how many grams of chemical X you should measure out. While this is a trivial calculation, it can sometimes be tedious to compute this by hand. This is exactly the problem addressed by this script.
