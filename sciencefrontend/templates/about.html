About ScienceFrontend

ScienceFrontend is an open-source project meant to ease the process of using scientific scripts. The frontend provides a web-accessible, dependency-free GUI for performing quick computations.

Other Info

ScienceFrontend is currently being written by Edward Kim. Development of ScienceFrontend is supported by the Canadian Light Source. As such, scientific tasks related to x-ray spectroscopy are currently the main focus for ScienceFrontend's capabilities.
